# Code quality

- [SonarQube](https://github.com/Sonarsource/sonarqube "sonarqube"): Static code checking tool to help check code defects, improve code quality, and speed of development by enabling potential or apparent errors in fast-track location codes.
- [Spotless](https://github.com/diffplug/spotless): Spotless is a multi-language code formatting tool that supports Maven and Gradle in the form of a Plugin.
- [CheckStyle](https://github.com/checkstyle/checkstyle "checkstyle"): Similar to Spotless, it can help programmers prepare Java code that meets the coding criteria.
- [PMD](https://github.com/pmd/pmd "pmd"): An extended multilingual static code analyzer.
- [SpotBugs](https://github.com/spotbugs/spotbugs "spotbugs"): FindBugs' successor. Static analysis tool to find errors in Java code.
- [P3C](https://github.com/alibaba/p3c "p3c"): Alibaba Java Sections PMD implementations and IDE plugin. This plugin is available on Eclipse and IDEA.

# Project Build

- [Maven](https://maven.apache.org/): A software project management and understanding tool. Based on the concept of the Project Object Model (POM), Maven can build, report, and document from a central information management project. Detailed description: [Maven Core Concept Summary](https://javaguide.cn/tools/maven/maven-core-concepts.html).
- [Gradle](https://gradle.org/): An open-source build automation tool that is flexible enough to build almost any type of software. Gradle makes very few assumptions about what you're going to build or how you're going to build it, which makes Gradle particularly flexible. Detailed description: [Gradle Core Concept Summary](https://javaguide.cn/tools/gradle/gradle-core-concepts.html).

# Invert

- [JADX](https://github.com/skylot/jadx): Command line and GUI tool for the generation of Java source code from Android Dex and APK files.
- [JD-GUI](https://github.com/java-decompiler/jd-gui): An independent GUI tool to display Java source code in CLASS files.

# Database

Database modeling

- [CHINER](https://gitee.com/robergroup/chiner): A free, open-source modeling tool for national production databases. The objective is to have a database-enrichment ecological and independent of the specific database, a database relationship model design platform. Formerly [EDMan](https://gitee.com/robergroup/pdman), a free alternative to PowerDesigner.

There are fewer open-source database modeling tools, and the following are some non-open-source database modeling tools (some of which require payment for use):

- [MySQL Workbench](https://www.mysql.com/products/workbench/): MySQL officially provides a visualization tool for database architects, developers, and DBAs. MySQL Workbench supports data modeling, SQL development, server configuration, user management, performance optimization, database backup and migration, and supports Windows, Linux, and Mac OS X platforms.
- [Navicat Data Modeler](https://www.navicat.com.cn/products/navicat-data-modeler): A powerful and cost-effective database design tool that helps users create high-quality conceptual, logical, and physical data models. Allows you to visualize the database structure, execute reverse or positive programs, import models from ODBC data sources, generate complex SQL/DDLs, and print models to file. Pay.
- [DbSchema](https://dbschema.com/): A powerful tool for the visualization of database design and management to support almost all relational and NoSQL databases. Pay.
- [dbdiagram.io](https://dbdiagram.io/home): It is a simple and free online ER mapping tool that creates models by creating codes designed for developers and data analysts. It generates a data model in a simple, custom language, supporting the functions of MySQL, PostgreSQL, SQL Server database DDL files, version history, online sharing, exporting pictures, or PDF. Dbdiagram.io provides a free version.

# Database Management

- [Chat2DB](https://github.com/alibaba/Chat2DB): A smart common database tool and SQL client for the Alibaba open source, supporting Windows, Mac local installation, and server-end deployment, Web page access. The traditional database client software Navicat, DBeaver, and Chat2DB, which combines AI capabilities, support the natural language generation of SQL and SQL performance optimization.
- [Beikeeper Studio](https://github.com/beikeeper-studio/beikeeper-studio): Cross-platform database management tool with high-value paint, supporting SQLite, MySQL, MariaDB, Postgres, CockroachDB, SQL Server, Amazon Redshift.
