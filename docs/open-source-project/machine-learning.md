Because Java is currently less used in the AI area, the associated open source projects are very few.

# Foundation

- [Spring AI](https://github.com/spring-projects/spring-ai): The artificial intelligence engineering application framework provides a Spring-friendly API and abstraction for the development of AI applications.
- [Spring AI Alibaba](https://github.com/alibaba/spring-ai-alibaba): An AI application development framework in the Java language designed to simplify Java AI application development and allow Java developers to develop AI applications as common applications using Spring.
- [LangChain4j](https://github.com/langchain4j/langchain4j): The Java version of LangChain is used to simplify the integration of LLM (Large Language Model) into Java applications.
- [Deeplearning4j](https://github.com/eclipse/deeplearning4j): Deeplearning4j is the first business-class, open source, distributed deep learning library for Java and Scala.
- [Smire](https://github.com/haifengl/smire): Machine Learning Library based on Java and Scala.
- [GdxAI](https://github.com/libgdx/gdx-ai): A framework of artificial intelligence, fully developed with Java, for game development using libGDX.

# In Action

- [springboot-openai-chatgpt](https://github.com/27405665/springboot-openai-chatgpt): A SpringCloud-based microservice structure has been docked with GPT-3.5, GPT-4.0, 100-degree words, Midjourney drawings, etc.
- [ai-beehive](https://github.com/hncboy/ai-behive): AI Hive, based on Java using Spring Boot 3 and JDK 17, supported by ChatGPT, OpenAI Image, Midjourney, NewBing, Words from the Heart, etc.
