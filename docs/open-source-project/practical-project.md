# Quick development platform

- [Snowy](https://gitee.com/xiaonuobase/snowy): The country's first state-owned, back-to-back and fast-development platform. Details: [5.1k! This is the strongest back-to-back separation I've ever seen!](https://mp.weixin.qq.com/s?__biz=Mzg2OTA0Njk0OA=&mid=2247534316&idx=1&sn=6993383774fc33ecda43c8c9d0a4039&chksm=cea10927f9d68031b862485c6be984ade5af233d871d488c22164a844678c067cd7&token=1464380539&lang=zh_CN#rd).
- [eladmin](https://github.com/elunez/eladmin): Backstage management system, back- and back-to-back, using modular development for projects, rights control using RBAC, supporting data dictionary and data permission management, supporting one key to generate back-to-back end codes and supporting dynamic paths.
- [RuoYi](https://gitee.com/y_project/RuoYi): RuoYi is based on SpringBoot's authorized management system. Easy to Read, Simple to Interface, which can be operated directly.
- [AgileBoot-Back-End](https://github.com/valarchie/AgileBoot-Back-End): Based on Ruoyi, extensive remodeling of the basic rapid development framework.
- [SmartAdmin](https://gitee.com/lab1024/smart-admin): A simple, easy-to-use low-code backstage solution.
- [EuBackend](https://gitee.com/zhaoeryu/eu-backend): Lightweight Rapid Development Platform based on SpringBoot.
- [RuoYi-Vue-Pro](https://github.com/YunaiV/ruoyi-vue-pro): RuoYi-Vue complete new Pro version, optimizes all functionality, supports data access, SaaS multi-tenant, Flowable workflow, tripartite login, payment, etc.
- [RuoYi-Vue-Plus](https://gitee.com/dromara/RuoYi-Vue-Plus): RuoYi-Vue completely new Plus version, rewritten all of RuoYi-Vue's functions, including S-Token, Mybatis-Plus, Jackson, SpringDoc, Hutool, OSS, etc.
- [pig](https://gitee.com/log4j/pig "pig"): RBAC Permission Management System based on Spring Boot + Spring Cloud + Outlook 2.
- [Guns](https://gitee.com/stylefeng/guns): Modern Java Application Development Foundation.
- [JeecgBoot](https://github.com/zhangdaiscott/jeecg-boot): A J2EE low-code rapid development platform based on code generator to support projects that generate front-end separation structures.
- [Erupt](https://gitee.com/erupt/erupt): Low-code whole stack framework, which uses Java Note to dynamically generate pages and back-office functions such as addition, deletion, modification, checking, and permission control.
- [BallCat](https://github.com/ballcat-projects/ballcat): A fully functional fast-development scaffold! In addition to the most basic privileges management, time job functions, additional support is provided for XSS filtering, SQL injection, data desensitization, etc.
- [JHipster](https://github.com/jhipster/generator-jhipster): Open-source application platform that can create SpringBoot+Angular/React in seconds.
