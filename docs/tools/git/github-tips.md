I've been using GitHub for more than six years, and today I've shared my little GitHub techniques, which I think are more useful, without reservation, to my little friends who are interested in JavaGuide.

# One key to generate GitHub CV & GitHub Annual

You can generate an online GitHub curriculum vitae via [https://resume.github.io/](https://resume.github.io/).

When I was in school, I put my personal information in an online GitHub resume. I think it'll make the interviewer feel like you're a pro. It'll give you some credit.

But if your GitHub doesn't have any items, don't include them in the résumé. The resulting effects are shown in the figure below.

![GitHub curriculum vitae](https://oss.javaguide.cn-2020-11/image20201108192205620.png)

Through this site <https://www.githubtrends.io/wrapped>, you can generate a GitHub personal annual report, which lists your project contributions for the year, the most commonly used programming language, and detailed contributions.

![GitHub annual report](https://oss.javaguide.cn/github/dootask/image-2021212460607457.png)

## Personalized GitHub Home Page

GitHub currently supports the self-defined presentation of some content on the personal home page. The results are shown in the figure below.

![Presentation of results on personalized home page](https://oss.javaguide.cn/java-guide-blog/image-20261612212259.png)

To do this, you just have to create a repository with the same name as your GitHub account and then customize the content of `README.md`.

This shows the self-defined content of `README.md` on your home page.

![Create a repository with the same name as your GitHub account](https://oss.javaguide.cn/java-guide-blog/image20201107111309341.png)

This one can play with flowers! For example, by [github-readme-stats](https://hellogithub.com/periodical/statistics/click/?target=https://github.com/anuraghazra/github-readme-stats), you can display dynamic GitHub statistics in README. The results are shown in the figure below.

![Dynamic GitHub statistics](https://oss.javaguide.cn/java-guide-blog/image-2020616221212426.png)

Not to mention the personalization of the front page, but the interested partners will study it themselves.
