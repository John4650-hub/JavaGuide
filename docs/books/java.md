# Java Foundation

**[Head First Java](https://book.douban.com/subject/2000732/)**

![Head First Java](https://oss.javaguide.cn/github/javaguide/books/image20204243035793.png)

The text of Head First Java is so easy and interesting that it's one of my favorite books from the early years of my programming. At the same time, this book is my Java Enlightenment Book. I was in the early stages of learning Java, and thanks to the help of this book, I was able to enter the Java language.

I think I'm in Java, and this piece can hold on, and this book has a lot of credit. A lot of friends around me have studied Java's first book.

A lot of young people would ask: **Is this book for new programmers?**

Personally, I think it's a good book for programmers; after all, it's part of the Head First series.

**[Java Core Technical Volume 1 + Volume 2](https://book.douban.com/subject/348994/)**

![Java Core Technical Volume 1 + Volume 2](https://oss.javaguide.cn/github/javaguide/books/image2022442417849.png)

These two books are very good. However, the contents of both books are substantial, and it would take time to read them all. I'm using these two books as tools; like when I write articles and have problems with Java's basics, I often look at them as references!

I bought them for my bedroom when I was in college. I flipped through them when I was free. The proposal is a bit Java-based, and then read it; the presentation is more in-depth, comprehensive, and highly recommended.

**[Java Programming Thoughts](https://book.douban.com/subject/21330190/)**

![Java Programming Thoughts](https://oss.javaguide.cn/github/javaguide/books/image2022424124893.png)

In addition, the author of the book published a new version of [On Java](https://book.douban.com/subject/35751619/) last year, which I recommend. It is an update and introduces three long-term support versions of Java (Java 8, 11, 17).

![On Java](https://oss.javaguide.cn/github/javaguide/books/on-java/6167760353_.pic_hd.jpg)

After all, this is the only technical book available on the market that contains three long-term support versions of Java.

**[Java 8 Operations](https://book.douban.com/subject/26772632)**

![Java 8 Operations](https://oss.javaguide.cn/github/javaguide/books/image202424203022625.png)

Java 8 is a landmark version and is now more commonly used by Java businesses. Some of the new features of Java 8, such as Lambda and Stream API, are necessary. In this case, I recommend **[Java 8 Fields](https://book.douban.com/subject/26772632)**.

**[The Logic of Java Programming](https://book.douban.com/subject/3013340/)**

![The Logic of Java Programming](https://oss.javaguide.cn/github/javaguide/books/image-2023072115350488.png)

A very low-key good book has deeper content than an introductory book. It's for beginners, but it's also for people to review Java's basics.
