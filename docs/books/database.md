## Database System Concept

[Database System Concept](https://book.douban.com/subject/10548379) covers the whole set of concepts of the database system; the knowledge system is clear and is a very classic learning material for the database system! Not a reference book!

![Image](https://oss.javaguide.cn/github/javaguide/books/image-20204009150441742.png)

## Database System Realization

If you also want to study the principles of MySQL, I suggest you read [Database System Realization](https://book.douban.com/subject/48388430/) first.

![Image](https://oss.javaguide.cn/github/javaguide/books/database-system-implementation.png)

Whether MySQL or Oracle, their overall structure is similar, differing in their internal realizations, such as data structure for the database index and how the storage engine works.

The book also has, in some places, a relatively poor translation, so it is advisable to read the English version or to use the English version.

The book "Database System Realization" is Stanford's educational material, in addition to a [Database System Basic Curriculum](https://book.douban.com/subject/3923575/) which is a pre-curricular course that allows you to enter the database.

# MySQL

Our website or AP data need to be stored using databases.

MySQL is more commonly used in general enterprise project development. If you want to study MySQL, you can see the following three books:

- **[MySQL Must Know](https://book.douban.com/subject/335490/)**: Very thin! It's perfect for MySQL reading. It's a great introduction.
- **[High Performance MySQL](https://book.douban.com/subject/3008813/)**: A classic work in MySQL! Learn MySQL! It is a step-by-step content that teaches you how to use MySQL better. There's both theory and practice! And if you don't have time to read it all, I suggest you take a good look at Chapters 5 and 6.
- **[MySQL Technology Inside](https://book.douban.com/subject/24708143)**: If you want to know more about the MySQL storage engine, this book is right for you!
