# Operating System

**Why Learn to Operate?**

**In terms of personal empowerment,** many ideas in the operating system, many classic algorithms, you can find their shadows in the various tools or frameworks that we use on a daily basis. For example, the system we've developed uses the same kind of cache (e.g., Redis) and the high-speed cache of the operating system. There are many types of high-speed caches in CPUs, but most of them are designed to address the unequal speed of CPU processing and memory processing. We can also view memory as a high-speed cache of memory, and when the program runs, we copy the memory of the memory, which increases the processing speed as the memory is processed much faster than the memory. Similarly, the Redis cache we use is designed to resolve the problem of uneven speed of process and access to regular relational databases. High-speed caches are generally subject to localism (2-8 principles) and the corresponding phase-out algorithm to ensure that the data in the cache are frequently accessed. The Redis cache that we usually use is done in many cases according to the 2-8 principles, and many of the elimination algorithms are similar to those in the operating system. Having stated the 2-8 principle, it was necessary to mention the hit rate, which is common to all the cache concepts. In short, how much of the data you want to access can be found directly in the cache. If you have a high hit rate, it usually shows that your cache design is reasonable and the system is processing relatively quickly.

**From an interview point of view**, especially in the case of school admissions, there is a great deal of research into the knowledge of operating systems.

**In short, learning operating systems enhances the depth of your thinking and understanding of technology, and the knowledge of the operating system is a requirement for interviews.**

If you want to learn about operating systems in a systematic way, the most authoritative book is **[The Introduction to Operating Systems](https://book.douban.com/subject/33463930/)**. You can add another **[Insight Computer System](https://book.douban.com/subject/1230413/)** to deepen your understanding of the nature of computer systems.
